# Gilang Akhil Azzam

A personal portfolio website built with React and JSX to showcase projects, skills, and personal information in a clean, modern, and professional way.

## Overview
This project is a multi-page personal portfolio website developed with a component-based structure. It is designed to be visually clean, responsive, and easy to maintain as the project grows.

## Features
- Built with React and JSX
- Component-based architecture
- Multi-page navigation
- Responsive layout
- Smooth animations and transitions
- Structured and scalable codebase
- Easy to customize and expand

## Tech Stack
- React
- JSX
- JavaScript
- CSS
